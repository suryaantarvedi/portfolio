# Surya Antarvedipalem — Portfolio Website

A clean, dark-themed personal portfolio for a Data Analyst fresher.

## Folder Structure

```
surya-portfolio/
├── index.html              ← Main HTML file (open this in browser)
├── css/
│   └── style.css           ← All styles (colors, layout, animations)
├── js/
│   └── main.js             ← Terminal typewriter + scroll reveal logic
├── assets/
│   └── pdf/
│       └── Surya_Resume.pdf  ← Place your resume PDF here
└── README.md
```

## How to Use

1. Place your resume PDF at: `assets/pdf/Surya_Resume.pdf`
2. Open `index.html` in any browser — no build step needed.
3. To deploy online, upload the entire `surya-portfolio/` folder to:
   - GitHub Pages (free)
   - Netlify (free, drag-and-drop)
   - Vercel (free)

## To Update Content

| What to change             | Where to edit       |
|----------------------------|---------------------|
| Name, bio, stats           | `index.html`        |
| Projects / certs           | `index.html`        |
| Colors, fonts, spacing     | `css/style.css`     |
| Terminal animation text    | `js/main.js`        |
| Resume file                | `assets/pdf/`       |

## Tech Stack

- Pure HTML5, CSS3, Vanilla JavaScript
- Google Fonts: Space Grotesk, Inter, JetBrains Mono
- No frameworks, no dependencies, no build tools
